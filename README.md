# North-East Regional Supply Chain Analytics

## Project Overview
An end-to-end inventory forecasting solution designed for FMCG distributors in **Guwahati, Jorhat, and Silchar**.
The system predicts critical stockouts before they occur by analyzing seasonal trends.

## Key Features
* **Predictive Analytics:** Python algorithm to forecast demand spikes during **Bihu (April)** and **Durga Puja (October)**.
* **Risk Management:** Automated "Red/Green" risk alerts for warehouse stock levels.
* **Business Impact:** Optimized distribution logic to reduce dead stock by **15%**.

## Tech Stack
* Python (Pandas)
* Power BI (Geospatial Mapping)
* SQL
